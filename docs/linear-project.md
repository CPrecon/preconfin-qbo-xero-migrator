# Linear Project: PRE-MIG-1 — QBO → Xero Migration Tool

## Goal

Build a production-ready public web application that lets users securely connect QuickBooks Online, analyze their accounting data, generate Xero-ready migration files, and receive a financial validation report.

## Milestones

1. Public Marketing Website
2. QuickBooks Connection
3. Canonical Accounting Model
4. Migration Engine
5. Validation Engine
6. CSV Generator
7. PDF Validation Report
8. User Dashboard
9. Lead Capture
